# Total records

This folder contains all the original records for each conservation unit.
If it's empty, you can populate it by running [analyses/treatOccs.R](analyses/treatOccs.R).
This will overwrite these files.
Each file is derived from the corresponding file in [results/total](results/total).
This is the same data as in [results/total](results/total), except ordered and in .csv format for human reading.
These files are not versioned because they are numerous and large.
