# Final checklists

This folder contains the top records for each taxon, for each conservation unit,
formatted according to the norms of [Catálogo de Plantas das UCs do Brasil](https://catalogo-ucs-brasil.jbrj.gov.br/).
You can populate it or update it by running [analyses/treatOccs.R](analyses/treatOccs.R).
This will overwrite these files.
Each file is derived from the corresponding file in [results/total](results/total).
